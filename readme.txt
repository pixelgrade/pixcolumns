== Description ==

Adds columns editing controls to the visual content editor (TinyMCE) based on the table editing UX.

Please note that the columns controls are contained in the “kitchen sink” toolbar, toggled with the last button on the first row of controls.

== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder 
`pixcolumns` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Show the toolbar in the editor by opening the "kitchen sink" (the last button in the first row)
